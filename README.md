Lyrica - платформа для синхронизации текстов к музыке. ты ищешь любую песню, смотришь её текст, синхронизированный со звуком слово в слово (иногда слог в слог). слова вылетают в точности под тайминги трека, а визуальный стиль подстраивается под настроение песни: цвета из обложки, скорость и характер анимации зависят от темпа и энергии инструментала.
как это работает (упрощенно):

-запускаешь песню в spotify/ЯМ/типа того
-Lyrica через системный api у виндоус определяет какой трек звучит
-скачивает трек, прогоняет через whisper (нейронка которая определяет когда звучит каждое слово)
-на экран выводится текст, синхронизированный ровно под музыку, с красивой анимацией
-если тайминги неправильные, то редактируешь их прямо в браузере
-результат сохраняется, и при следующем прослушивании всё уже готово (кэширование буууэ)

атмосфера:

-для чего то жесткого текст резко вылетает крупными буквами
-для ambient/lofi трека текст плавно тает, небольшой размер (или большой, но мягкий), мягкая палитра
-всё это генерируется автоматически по audio features трека
